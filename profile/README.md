lunar landing 社区致力于让 "WEB" 开发变得简单优雅，让更多的开发者能够零距离和项目作者交流或学习他们优秀的项目作品。

但如有发现 lunar landing 社区孵化项目或所属作者有从事违法活动或危害他人信息安全行为，则 lunar landing 组织将立即下架该项目并收集其违法行为提供至相关执法人员并积极配合调查，同时 lunar landing 组织将该作者和所加入项目列入制裁黑名单并永不再接纳。

**免责申明： 所孵化项目的一切违法行为皆和 lunar landing 组织无关。**
